## <p style="text-align: center;">AWS Partner: AWS Serverless GIF factory</p>

*About the Labs*

In this workshop you will create a *Serverless* GIF factory application on AWS. You will work with the following AWS services:

* S3
* IAM
* DynamoDB
* SNS
* CloudWatch Logs
* Elastic Transcoder
* Lambda
